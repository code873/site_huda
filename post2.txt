<h3>Thowing Windows & Take Linux</h3>
Ya, di sini aku mau cerita terkait hijrahku dari Windows ke Linux. Sebelumnya aku pernah cerita di post 1 bahwa dulu waktu SMA aku udah menjadi pengguna linux, sehingga aku tidak newbie-newbie amat lah di OS satu ini. Tetapi, 9 tahun bukan waktu yang singkat ya, aku jadi banyak lupa dengan command linux.<br>
Oke, aku mulai dari kenapa aku leaving Windows. Pada bulan Juli 2025, aku semakin gelisah aja, stres, nggak mau bertahan gini-gini terus menjadi pengajar bimbel. Ya, saat itu aku sedang bekerja di salah satu bimbingan belajar di Jogja. Walaupun tempat kerja nya nyaman tetapi aku udah mulai tidak menikmati pekerjaan sebagai "pengajar siswa" selain masalah gaji juga. Kemudian, aku carilah alternatif karir yang masih agak relate dengan math tetapi bukan mengajar. Sempat aku mau terjun ke dunia Ekonomi, Keuangan, atau Akuntansi, hingga sempat beberapa hari aku luangkan waktu untuk mencari referensi dan mempelajari ilmu-ilmu yang dibutuhkan untuk bekerja sebagai Staf Keuangan atau Akuntan. Jelas mulai dari NOL, karena aku tidak tau apa-apa awalnya. Singkat cerita, pembelajaranku tidak berjalan baik, sehingga timbul pikiran untuk kembali ke dunia komputer.
Kemudian, di awal juli aku memutuskan untuk belajar komputer lagi. Di situ aku mikir, kalo belajar komputer apakah sekalian aku belajar linux lagi ya?, tapi harus install linux, dll. Beberapa hari aku memikirkan, dan akhirnya kuputuskan untuk install linux di laptop ku karena sudah tidak betah lagi dengan ke-lemot-an dan akses yang terbatas dari windows. Aku mulai dengan Ubuntu 24.04 LTS mengikuti tutorial di youtube (karena sudah 9 tahun, jelas sudah lupa cara install). Sebelum aku install ubuntu, data-data yang tersimpan di windows tidak aku backup karena aku belum punya penyimpanan eksternal dan kalau upload ke google drive cukup lama karena data-nya berat. Alhasil beberapa berkas penting yang tidak berat aku backup ke HP dan yang lain aku ikhlasin aja. Akhirnya, ku install ubuntu sekaligus mem-format windows dengan niatan komputer ku ini bersih dari hal-hal yang memberatkan, sementara aku bukan orang yang suka sistem dualboot.
Singkat cerita ubuntu 24.04 LTS sudah berhasil terpasang dan mulailah aku belajar command terminal dan seterusnya. Adaptasi dulu dengan OS kali ini. Beberapa hari setelah menggunakan ubuntu versi ini, aku menemukan bug, specifically saat update dan upgrade, apt/sources.lists nya error. Rasakanya bukan seperti masalah yang harus mengganti mirror jaringan karena keterangan error menyatakan bahwa release ubuntu tidak terupdate atau seperti apa saya agak lupa. Nah, dengan bug seperti itu, untuk orang yang baru hijrah ke linux kan cukup mengganggu ya. Akhirnya, aku coba install ulang ubuntu ini, kalo tidak salah sampe 3x mungkin, tetapi selalu mengeluarkan bug yang sama. Jadilah kemudian aku coba search distro linux pengganti ubuntu. Kalo bisa distro lain jangan ubuntu dengan beda versi aja. Ingatlah kemudian dengan yang namanya 'debian'. Aku cari file installernya di web resminya, tapi hanya tersedia untuk beberapa versi terakhir, seperti versi 12 (bookworm) dan 13 (trixie). Di youtube, ada channel bagus yang menjelaskan tutorial install debian 11 dan tampilan versi ini cukup bagus dibandingkan dengan versi 12 dan 13. Terbawalah aku kemudian ke suatu website yang di sana menampung semua installer debian secara lengkap, sehingga dapatlah aku file debian 11 (bullseye) xfce.
BTW, sebelum melangkah lebih jauh, aku mau terangkan dulu apa itu linux, distro ubuntu, dan distro debian. Linux adalah OS yang open source (terbuka kode-nya), artinya kita bisa konfigurasi OS ini semau kita. Beda dengan OS windows yang serba tertutup aksesnya, sehingga kita hanya bisa menggunakan tanpa bisa konfigurasi dengan leluasa. Selain itu, linux juga free. Kita tinggal download aja file installer nya di internet, kemudian install sendiri. Nah, awalnya linux dibuat oleh mahasiswa ilmu komputer asal Finland bernama Linus Torvalds. Dia pernah bilang alasannya membuat linux karena ketidakbebasan akses terhadap OS yang ada pada zaman dia kuliah. Orang ini terkenal akan fleksibilitas dan kebebasannya, makanya ia membuat linux bisa gratis dan open source buat semua orang. Yang paling plot twist adalah dia menyebutkan bahwa membuat linux ini sebenarnya cuma hobi dan nggak bakal menyangka bakal menjadi OS paling penting dan banyak digunakan bahkan sampai ke sistem besar seperti NASA. Linus juga membebaskan OS ini dipakai siapa saja dan boleh dikembangkan semaunya (yang penting untuk tujuan kebaikan). Setelah para programmer banyak yang minat, mereka mencoba develop linux sedemikian rupa sehingga memunculkan jenis-jenis (distro) linux dengan kegunaan/manfaat khusus, diantaranya ubuntu, debian, slackware, fedora, redhat, kali linux, arch linux, dan masih banyak lagi. Sekarang, akan ku bahas ubuntu dan debian. Dalam urutan silsilah, sebenarnya debian ini adalah bapak dari ubuntu, which is debian ini lebih tua dari ubuntu. Adapun kelebihan dan kekurangan dari keduanya: debian disukai karena ke-stabilan sistem nya, lebih stabil dari pada ubuntu. Stabil di sini maksudnya adalah sistem dan paket data dari debian hampir benar-benar tidak ada bug karena memang debian selalu mengecek betul paket tertentu sebelum dirilis. Akibatnya, developing nya agak lama. Tetapi, dari perlakuan ini kan yang membuat debian itu berdiri kokoh. Inilah linux yang cocok denganku. Di sisi lain, ubuntu lebih user-friendly dari pada debian, atau bisa dikatakan bahwa ubuntu itu windowsnya linux. Paket aplikasi yang windows banget lebih disupport ubuntu daripada debian. Jadi, pemilihan mana distro linux yang tepat bergantung pada apa yang sebenarnya kamu butuhkan, untuk kebutuhan bisnis bisa pakai distro fedora, untuk kebutuhan cybersecurity bisa pakai kali linux, dan lain-lain.
Akhirnya, aku berhasil install debian 11 dan menemukan kebahagiaan di sana. Merasa lebih baiklah intinya, bug yang aku temukan di ubuntu kemarin tidak kutemukan di debian ini. Bersyukurnya debian 11 batas supportnya masih lama (2028 kalo tidak salah), sehingga tidak perlu khawatir dengan paket-paket yang kadaluarsa. Tetapi, aku punya fun fact bahwa penyakitnya pengguna linux adalah kita gampang tergoda oleh setan untuk ganti-ganti distro. Beberapa minggu kemudian, aku kena godaan itu karena aku sempat merasakan kok agak lemot ya padahal udah versi xfce. Terus aku mikir, ada satu distro yang sangat ringan bernama slackware. Ya, benar, dia sangat ringan tetapi umurnya dia udah tua banget, dia lebih tua dari debian malah. Banyak pertimbangan buat install dia, tetapi dengan rasa curiosity yang suangat besarrr ini,, akhirnya aku install slackware versi 12 (rilis tahun 2016, tua banget boy). Penginstalannya cukup rumit, benar-benar manual banget, tapi aku bisa aja sih alhamdulillah. Singkat cerita, akhirnya terinstall lah slackware 12. Nah, inilah yang kemudian aku merasa sangat menyesal, bahwa karena saking tua nya dia, banyak paket aplikasi yang tidak disupport oleh dia. Dan yang paling parah lagi adalah aku tidak menemukan mirror jaringan yang tepat untuk distro ini. Akibatnya, aku gak bisa melakukan update dan upgrade dong. Yaudah deh, kukembalikan lagi ke debian 11. Gila gak kalian bacanya,, itulah penyakit dari menggunakan linux. Emang gaada OS yang bisa memberikan kita kepuasan maksimal secara terus-menerus.
Dengar-dengar bahwa debian support upgrade sistem ke versi atas. Wah ini kesempatan aku untuk mencoba debian 12 tanpa mendownload installer debian 12 dan menginstallnya. Ternyata caranya cukup mudah, hanya dengan melakukan konfigurasi file apt/sources.list/ lalu melakukan update dan upgrade paket-paket debian 12. Hanya saja proses memakan waktu kurang lebih 3 jam. Ceritanya, aku udah berhasil upgrade ke versi 12 tetapi setelah beberapa minggu, debian 12 ini berat sekali. Walah yaudah, akhirnya banget aku install ulang debian 11 (karena debian tidak support downgrade dari 12 ke 11). Tetapi, insyaAllah versi bullseye ini akan aku pertahankan sampai batas LTS berakhir selama tidak menemukan bug yang tidak bisa aku selesaikan.
Begitulah story perjalananku menggunakan linux lagi.. hal baik silakan diambil dan hal buruk silakan diabaikan..
Terima kasih sudah membaca!
